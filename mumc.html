Task Details: Website Development

The Melbourne University Mountaineering Club (MUMC) is a club that organises hill walking, mountaineering, and climbing trips for its members. It produces a club newsletter each month, with articles about club matters and club trips.

The club requires an interactive brand-building website, which will enable their members to obtain information about new trips, costings, and view images of expeditions. The club has had a backend developer prepare the website files for you. To make the website work, you must ensure you follow the client’s specification exactly!

Task 1: Logo development – Total 2 Marks

The MUMC would like a logo. The club’s president, Munro Walker, has set out the following requirements for the logo:

  • To be created from scratch
  • To be a pixel size of 130 pixels high × 130 pixels wide
  • To include the club initials MUMC in black
  • To include a simple motif to represent mountains
  • To have a transparent background
  • To be stored in a common file format

(a) Implement the logo and provide annotated evidence to show how the requirements were met.

The requirements were met for the logo to fit the client’s needs, being the correct size and fitting description. The image size is 130 pixels by 130 pixels.

Task 2: Website – Total 33 Marks

Web Banner

Munro Walker, the club’s president, would like to have an animated banner for the website. The banner must use the four photos that have been provided: Dawn.jpeg, Summit.jpeg, Ridge.jpeg, and Plateau.jpeg.

  • The banner size will be 1000 pixels wide by 150 pixels high
  • The banner should be animated so that it cycles through the four images one after the other
  • The images should appear to fade in and fade out
  • The banner sequence should repeat automatically every 30 seconds
  • The club name should be present and clearly visible throughout
  • The MUMC club logo (created in task 1) should be included on the right-hand side of the banner

The requirements for the banner were met as the different images were cycled through to create a visually appealing header for the website.
